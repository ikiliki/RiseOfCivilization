# Feature Documents

Feature specs created by the Product agent live here.

## Format

Each feature: `docs/features/<feature-name>.md`

- Purpose, scope, user flows
- Acceptance criteria
- Examples, Figma links
- Technical Architecture (from team-lead)
- Technical Specification (from team-lead)
- Implementation Tasks

## Workflow

1. `/product` creates initial design doc
2. `/team-lead` adds architecture and technical spec
3. `/coder` implements from spec
